# Drop village bundles here

Download a bundle from the site's **Get started** page and unzip it into this folder, so you have:

```
data/34855\_vadnerbhairav\_chandavad\_nashik/
  input.geojson
  imagery.tif
  boundaries.tif
  example\\\_truths.geojson



data/Malatavadi/
input.geojson
imagery.tif
boundaries.tif
example\\\_truths.geojson
```

Then, from the kit root: `uv run quickstart.py data/<village\\\_slug>`
